# GävleGuiden

GävleGuiden är en modern och responsiv webbplats som guidar dig till Gävles mest intressanta platser och sevärdheter. Projektet är helt statiskt och passar både invånare och besökare som vill upptäcka mer av staden.

## Funktioner

- **Utforska platser:** Läs om Boulognerskogen, Gävlebocken, Järnvägsmuseet, Fängelsemuseet, Konserthuset och Fjärran Höjder.
- **Bilder & kartor:** Varje plats har bilder och inbäddad Google Maps-karta.
- **Responsiv design:** Snygg och användarvänlig på både mobil och dator.
- **Ljust/mörkt läge:** Växla tema med ett knapptryck.
- **Om oss & kontakt:** Information om projektet och kontaktmöjlighet.

## Filstruktur

```
src/
  index.html           # Startsida
  om-oss.html          # Om projektet
  kontakt.html         # Kontaktinformation
  css/styles.css       # Stilmall
  js/main.js           # JavaScript för tema och årtal
  platser/             # Undersidor för varje plats
    boulognerskogen.html
    gavlebocken.html
    jarnvagsmuseet.html
    fangelsemuseet.html
    konserthuset.html
    fjarran-hojder.html
raw_img/               # Bilder till platserna
```

## Kom igång

1. Klona eller ladda ner projektet.
2. Öppna `src/index.html` i din webbläsare.
3. Navigera mellan sidorna via menyn.

Ingen server eller installation krävs – allt fungerar direkt i webbläsaren.
